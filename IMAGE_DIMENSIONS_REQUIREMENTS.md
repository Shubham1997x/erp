## üìù **Notes for Screenshot Taking**

### For 1200 x 700 px images (Featured Screenshots):
- These are the main feature images
- Take full-screen or window screenshots
- Ensure good quality and clarity
- Landscape orientation
- Can be cropped/resized to exact dimensions later

### For 600 x 400 px images (Order Steps):
- Smaller card images
- Can be cropped from larger screenshots
- 3:2 aspect ratio

### For 200 x 200 px images (Testimonials):
- Square format
- Should focus on face/head area
- Will be displayed as circular avatars

### For 200 x 100 px images (Logos):
- Wide landscape format (2:1 ratio)
- Company logos or brand images
- Should be clear and readable

## üéØ **Recommended Screenshot Tools**

- **Windows:** Snipping Tool, Windows + Shift + S, or ShareX
- **Mac:** Cmd + Shift + 4, or built-in screenshot tool
- **Browser Extensions:** Full Page Screen Capture, Awesome Screenshot
- **Online Tools:** Canva, Photopea for resizing/cropping
